# Запуск Fleet AI Project через Docker

Этот проект можно легко запустить, используя Docker и Docker Compose. Это изолирует все зависимости и позволяет быстро развернуть приложение в любой среде.

## Требования
- Установленный **Docker**
- Установленный **Docker Compose** (обычно поставляется вместе с Docker Desktop)

## Шаги для запуска

1. **Откройте терминал** и перейдите в корневую директорию проекта (там, где находится файл `docker-compose.yml`).
   ```bash
   cd fleet_ai_project
   ```

2. **Соберите и запустите контейнеры**:
   Для сборки образа и запуска сервиса в фоновом режиме выполните:
   ```bash
   docker-compose up -d --build
   ```
   *(Если у вас Docker Compose V2, команда может выглядеть как `docker compose up -d --build`)*

3. **Проверка работы**:
   После успешного запуска приложение будет доступно по следующим адресам:
   - **Пользовательский интерфейс (UI / Dashboard & Chat)**: [http://localhost:8000/](http://localhost:8000/)
   - **Документация API (Swagger UI)**: [http://localhost:8000/docs](http://localhost:8000/docs)
   - **Документация API (ReDoc)**: [http://localhost:8000/redoc](http://localhost:8000/redoc)

4. **Остановка приложения**:
   Чтобы остановить приложение, находясь в директории проекта, выполните:
   ```bash
   docker-compose down
   ```

## Дополнительно
- База данных (SQLite) монтируется как том (`volume`) в `./fleet.db`, поэтому данные сохраняются даже после перезапуска или удаления контейнера.
- Для просмотра логов приложения используйте команду:
  ```bash
  docker-compose logs -f
  ```
